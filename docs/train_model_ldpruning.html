Introduction

We performed same analysis with this part. But the data are kind of different. After dividing the data into training part and testing part, we perfpormed LDpruning for the training data using Plink. The window size of ldpruning is 50kb, the step size is 5kb, the VIF is 5 (1/(1-r2), r2=0.8). After ldpruning, the average number of variants in training data decreases from 103,809 to 31,834.

I’m kind of confused here, after ldpruning, some singnals have been omitted. I expected to get lower r-squared for each LV, but it seems that I’m totally wrong(see the scatter plot below). I plotted the r-squared obtained from all variants with p-val < 0.01 vs r-squared obtained after ldpruning. Why is this so?

Variants with non-zero coefficients
