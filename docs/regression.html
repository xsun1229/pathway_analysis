Introduction

In this part, we will use lasso regression to train a prediction model. The dependent variables here are LVs extracted from pwy-plier-nochrx and the independent variables are genotype of certain variants. (LVs ~ variant genotype)

Material and Methods

  1. We used whole blood gene expression data from GTEx including 670 samples. In genptype data, there are 10,403,249 variants in total (variants in chrX have been excluded).
  2. Performed association test between the 128 LVs and all SNPs using Plink. We corrected the results for genotype PCs (We used Plink to compute 10 PCs and used them as corvariates in association test).
  3. From 2, we selected all SNPs with p < 1E-02. For each LV, we can get a set of around 100,000 variants.
  4. Trained Lasso/Elastic net using these SNPs on the training data(80% of the samples) and evaluated the performance in the testing data(20% of the samples).

##Results

Heritability vs R-squared

This figure shows the prediction performance (R2 of Lasso/Elastic_net regression on training set in red) in comparison to LV heritability estimates (black; standard errors in gray). Performance was assessed using tenfold cross-validation in the GTEx whole-blood cohort (on training set, n = 536) with the elastic net and lasso models.
