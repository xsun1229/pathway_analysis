Introduction

In this part, we will use lasso/elastic_net regression to train prediction models. The dependent variables here are LVs extracted from pwy-plier-nochrx and the independent variables are genotype of certain variants. (LVs ~ variant genotype)

Material and Methods

  1. We used whole blood genotype data from GTEx including 670 samples. In genptype data, there are 10,403,249 variants in total (variants in chrX have been excluded). There are some missing values in our dataset. We used mean imputation (substitue the missing values with mean value of the corresponding variants) to impute the missing value.

The missing data pattern of our dataset. x-axis represents the proportion of missing value for each variant. y-axis represents the number of the variants.

  1. We splited the data set into training(80%, 536 samples) and test(20%, 134 samples) set.

  2. In the training data, performed association test between the 128 LVs and all SNPs using Plink. We corrected the results for genotype PCs (We used Plink to compute 10 PCs and used them as corvariates in association test).

  3. From 3, we selected all SNPs with p < 1e-02. For each LV, we can get a set of a bit less than 100,000 variants.

  4. Trained Lasso/Elastic net using SNPs from step4 in the training data and evaluated the performance in the testing data.
