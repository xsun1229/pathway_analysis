Introduction

In this part, we used ‘flashr’, methods for matrix factorization based on Empirical Bayes Matrix Factorization, to extract factors from IDIN gene expression matrix (same with the matrix in PLIER_IDIN).We also performed factor analysis find if any factor associates with the two SNPs ( rs9557207 and rs11616269)

Material and Methods

  1. We used workflow showed in ‘flashr’ to extract factors.
f = flash(Y)    ### Y is the IDIN gene expression matrix.
ldf = flash_get_ldf(f)
  1. Then, we used the backfitting algorithm to improve the fit (but it takes longer).
f.b = flash(Y, f_init = f, backfit=TRUE, greedy=FALSE)
ldf.b = flash_get_ldf(f.b)
  1. Perform association test between the facotrs and two SNPs.

Results

We obtained 62 factors using FLASH.

Association test

We performed linear regression for factors ~ snp.

  ### k indicates the kth factors
  y <- as.numeric(factors[,k])   ## kth factors
  x <- as.numeric(geno_2snp[,1])   ## genotype of rs11616269
  
  pval_snp1[k] <- summary(lm(y ~ x))$coefficients[2,4]  ## pval of F-statistics in y ~ x regression
  rsq_snp1[k] <- summary(lm(y ~ x))$r.squared    ## r-squared of the regression
